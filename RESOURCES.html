Agent Architecture Resources

Primary Codebases

Coding Agents

  • openai/codex — GitHub. OpenAI's open-source coding agent CLI in Rust (Apache 2.0). ~90k stars. Multi-provider, sandboxed execution, multimodal input.
  • anomalyco/opencode — GitHub. MIT-licensed terminal coding agent. TypeScript server + Go TUI. Client/server architecture, built-in LSP, ACL permissions, plugin system. ~26k stars.
  • badlogic/pi-mono — GitHub. Minimalist terminal coding harness by Mario Zechner. Four primitive tools (read/write/edit/bash). Philosophy: "primitives, not features."
  • claude-code (mirror) — GitHub. Claude Code source recovered from npm source map leak. ~1,900 TypeScript files, ~512k LOC.

General Agents

  • openclaw/openclaw — GitHub. Personal AI assistant, Node/TypeScript. 25+ messaging channels, multi-agent routing, isolated workspaces.
  • NousResearch/hermes-agent — GitHub. Self-evolving AI agent, Python. ~23k stars. Three-layer memory, skill creation from experience, sub-agent parallelism.
  • HKUDS/nanobot — GitHub. Ultra-lightweight personal AI agent, Python, ~3.5k LOC. Multi-channel, MCP integration, persistent memory.
