学习记录 0002：两种核心循环架构的发现

2026-06-13

对 7 个生产级 agent 代码库的分析揭示了两种主导的循环架构：嵌套双层循环（nested two-loop，见于 pi-mono、openclaw）与扁平 while-true 循环（flat while-true，见于 claude-code、hermes-agent）。

关键驱动因素是恢复复杂度（recovery complexity）——扁平循环将级联回退路径集中在一处，一目了然；嵌套循环则在恢复逻辑简单时提供更清晰的关注点分离。

第三种模式（显式状态机，如 nanobot 的 7 状态 TurnState）虽然可行，但较为少见。

另识别出：batch 与 streaming 两种 tool 执行方式，作为与循环结构正交的次级维度。
