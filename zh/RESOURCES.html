Agent 架构资源

主要代码库

编程 Agent

  • openai/codex — GitHub。OpenAI 的 Rust 语言开源编程 Agent CLI（Apache 2.0）。约 9 万星。支持多提供商、沙盒执行、多模态输入。
  • anomalyco/opencode — GitHub。MIT 许可的终端编程 Agent。TypeScript 服务端 + Go TUI。客户端/服务端架构，内置 LSP，ACL 权限系统，插件系统。约 2.6 万星。
  • badlogic/pi-mono — GitHub。Mario Zechner 的极简终端编程框架。四个原始工具（读/写/编辑/bash）。核心理念：「原始能力，而非功能堆砌。」
  • claude-code (mirror) — GitHub。从 npm source map 泄露中恢复的 Claude Code 源码。约 1900 个 TypeScript 文件，约 51.2 万行代码。

通用 Agent

  • openclaw/openclaw — GitHub。个人 AI 助手，Node/TypeScript。25+ 消息通道，多 Agent 路由，隔离工作区。
  • NousResearch/hermes-agent — GitHub。自我进化的 AI Agent，Python。约 2.3 万星。三层记忆系统，从经验中创建技能，子 Agent 并行。
  • HKUDS/nanobot — GitHub。超轻量个人 AI Agent，Python，约 3500 行代码。多通道，MCP 集成，持久化记忆。
